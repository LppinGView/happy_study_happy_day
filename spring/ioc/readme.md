## 自己怎么手搓一个IOC容器出来？

Ioc容器，管理bean的生命周期，使用一个map对容器进行装载，bean初始化后，可以通过Ioc容器拿到注册的bean
同时通过代理，可以在管理bean的同时，对bean进行加强等操作


1. 支持bean的注册、获取、及容器的释放等操作；
2. 支持构造器注入(也可以扩展为xml方式)
3. 通过注解实现可插拔 动态扩展
4. 基于jdk动态代理，扩展bean的注册过程；
